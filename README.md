# AWS EC2 Flask Hello World Deployment

Simple Flask web app deployed to AWS EC2 with automated Bash script.

## Tech Stack
- Python + Flask
- AWS EC2 (free tier)
- Git for version control
- Bash scripting for automation

## How It Works
1. Launch EC2 instance (Amazon Linux 2, t2.micro).
2. Add user data (paste the deploy.sh content) for auto-setup.
3. Access via public IP.

## Learning Outcomes
- Automated deployment
- Cloud infrastructure basics
- Scripting in Bash
